To use this code, you will need:
    1. The Dependencies listed in the requirements.txt
    2. A GeminiAPI key (GOOGLE_AI_KEY) provided by Google Cloud Services
    3. A sense of humor
    4. Low expectations

Edit the KEYS.env and replace (GOOGLE_AI_KEY) with your key
GitIgnore is set to delete any insertions into the KEYS.env or a .venv if you choose to make one

Start or Open your Terminal
Type these commands into your Terminal:
    pip install install flask
    pip install -q -U google-generativeai
    pip install python-dotenv
    
These will be needed to resolve import conflicts

The server will automatically run on application execution, and provide the url to the App webpage
To close the server, use Ctrl+C
