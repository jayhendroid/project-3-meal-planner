To use this code, you will need:
    1. The Dependencies listed in the requirements.txt
    2. A GeminiAPI key (GOOGLE_AI_KEY) provided by Google Cloud Services
    3. A sense of humor
    4. Low expectations

Edit the KEYS.env and replace (GOOGLE_AI_KEY) with your key
GitIgnore is set to delete any insertions into the KEYS.env or a .venv file if you choose to make one

Start or Open your Terminal
Type this command into your Terminal:
    pip install -q -U google-generativeai

The server will automatically run on application execution
